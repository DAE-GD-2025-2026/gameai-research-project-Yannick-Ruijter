## What are Octrees
To make it easier to understand octrees, i'll first explain the origin. Enter **QuadTrees**! In definition, a quadtree is just a tree-structure where every node as either no or exactly 4 children. Now you might be confused
as to why this is usefull at all. Well this structure was first used around 1934 for the Whitney covering lemma to assert that there is a certain type of partition of an open set in a Euclidian space. It was only later in 
1974 that this was called a Quadtree. Now I already gave a hint to what the use of this structure could be. It's used for spatial partitioning.
